₊★Materiales★₊

  • Canicas (3)
  • Hilo de pescar
  • Palitos de baja lengua (10-20)
  • Una base (en este caso usamos un joyero viejo)
  • Silicona caliente
  • Decorativos (opcional)

₊★Construcción★₊

  1. Pegar los baja lenguas para formar la estructura del techo y las paredes
  2. Pega todo esto a la base
  3. Envolver las canicas con el hilo de pescar
  4. Pegar los hilos colgantes de forma recta al techo
  5. Por último decorar
